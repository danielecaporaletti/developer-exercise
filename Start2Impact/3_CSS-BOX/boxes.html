Headings Are Block Elements

Paragraphs are blocks, too. However, <em> and <strong> elements are not. They are inline elements.

Block elements define the flow of the HTML document, while inline elements do not.

The “CSS box model” is a set of rules that determine the dimensions of every element in a web page. It gives each box (both inline and block) four properties:

  • Content - The text, image, or other media content in the element.
  • Padding - The space between the box's content and its border.
  • Border - The line between the box's padding and margin.
  • Margin - The space between the box and surrounding boxes.

I margini e il riempimento possono ottenere la stessa cosa in molte situazioni, rendendo difficile determinare quale sia la scelta "giusta". I motivi più comuni per cui dovresti scegliere uno piuttosto che l'altro sono:

  • Il padding di un box ha uno sfondo, mentre i margini sono sempre trasparenti.
  • La spaziatura interna è inclusa nell'area di clic di un elemento, mentre i margini non lo sono.
  • I margini collassano verticalmente, mentre il padding no (ne parleremo più approfonditamente nella prossima sezione).

L'unica vera differenza tra a "div" e "span" è che il primo è per i contenuti a livello di blocco mentre il secondo è pensato per i contenuti in linea.
